ASTRA

Project Description:

Astra is a cutting-edge robot designed to enhance warehouse operations by automating product delivery. It uses advanced line-following technology to navigate along a predefined black line, ensuring precise movement throughout the facility. Equipped with Bluetooth connectivity, Astra communicates with external devices, enabling real-time data exchange by scanning QR codes and verifying product details with the warehouse system. After verifying data, Astra autonomously delivers products to designated locations, avoiding obstacles and optimizing routes. By combining automated navigation with accurate data verification, Astra significantly boosts warehouse efficiency and productivity.

Astra Usage in different sectors:

1) Manufacturing: It can scan barcodes or QR codes on parts and products, verifying them with the database to ensure accurate inventory management. Astra can transport raw materials between workstations on a predefined path, streamlining the assembly line process.

2) Healthcare: Astra can be programmed to follow paths in hospitals, delivering medicine or medical supplies between departments or to patient rooms. It can be used to transport medical waste securely from wards to designated disposal areas.

3) Logistics: In distribution centers, Astra can be used to sort and deliver packages by following lines to specific areas or loading docks.It can verify package information and ensure they are placed in the correct shipment areas based on the scheduled routes.

4) Airports: Astra can assist in transporting baggage from one terminal to another or from the check-in counter to loading areas. It can navigate through airport hangars or terminals delivering tools and supplies to different departments.

5) Hospitality: In hotels, Astra can follow predefined paths to deliver food, drinks, or amenities to guest rooms efficiently.It can assist by delivering towels, cleaning supplies, or other items to different floors or sections.

6) Agriculture: Astra can navigate through greenhouse paths, monitoring crops, delivering fertilizers, or collecting sensor data to maintain optimal growing conditions.It can be programmed to transport harvested crops from the field to storage areas or processing stations.

7) Automotive: Astra can be used in automotive assembly plants to transport car parts between workstations, reducing human labor and improving efficiency. It can scan and verify parts as they move through the production line, ensuring quality checks at each stage.

Circuit Diagram

Code

                #define LS 10      // left sensor
                #define RS 9       // right sensor
                #define LM1 7      //motor Driver in1
                #define LM2 6     //motor Driver in2
                #define RM1 5      //motor Driver in3
                #define RM2 4      //motor Driver in4
                
                void setup()
                {
                  pinMode(LS, INPUT);
                  pinMode(RS, INPUT);
                  pinMode(LM1, OUTPUT);
                  pinMode(LM2, OUTPUT);
                  pinMode(RM1, OUTPUT);
                  pinMode(RM2, OUTPUT);
                
                }
                void loop()
                {
                  
                  
                  if(!digitalRead(LS) && !digitalRead(RS))  
                  {
                       // FORWARD
                    digitalWrite(LM1, HIGH);
                    digitalWrite(LM2, LOW);
                    digitalWrite(RM1, HIGH);
                    digitalWrite(RM2, LOW);
                  }
                  
                  if((digitalRead(LS)) && !digitalRead(RS))     
                  {
                      // Left turn
                    digitalWrite(LM1, LOW);
                    digitalWrite(LM2, LOW);
                    digitalWrite(RM1, HIGH);
                    digitalWrite(RM2, LOW);
                  }
                  
                  if(!digitalRead(LS) && (digitalRead(RS)))     
                  {
                        // turn right
                    digitalWrite(LM1, HIGH);
                    digitalWrite(LM2, LOW);
                    digitalWrite(RM1, LOW);
                    digitalWrite(RM2, LOW);
                  }
                  
                  if((digitalRead(LS)) && (digitalRead(RS)))    
                  {       //Stop
                    digitalWrite(LM1, LOW);
                    digitalWrite(LM2, LOW);
                    digitalWrite(RM1, LOW);
                    digitalWrite(RM2, LOW);
                  }
                }
